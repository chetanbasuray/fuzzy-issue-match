# Roadmap

The product vision is to prevent duplicate issue noise by recommending already-open or already-closed issues with confidence scores.

## Milestones

1. `M1`: Core text preprocessing and Levenshtein scoring.
2. `M2`: Semantic similarity scoring and weighted ranking.
3. `M3`: GitHub Action integration and auto-comment workflow.
4. `M4`: Precision/recall benchmarking and tuning dataset.
5. `M5`: Scale and caching improvements for large repos.
6. `M6`: Public examples, guides, and adoption playbook.
